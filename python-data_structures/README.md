# 0-print_list_integer

## Description
This project contains a Python function that prints all integers of a list, one per line.

## Function

### `print_list_integer(my_list=[])`
- **Arguments**:  
  - `my_list` (optional): A list of integers. Defaults to an empty list.
- **Behavior**:  
  Prints each integer in the list on a separate line using `str.format()`.  

## Example

```python
my_list = [1, 2, 3, 4, 5]
print_list_integer(my_list)
