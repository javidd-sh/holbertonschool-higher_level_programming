===========================
Tests for MyList class
===========================

Import the class:
    >>> MyList = __import__('1-my_list').MyList

--- Basic usage ---

Empty list:
    >>> my_list = MyList()
    >>> my_list.print_sorted()
    []

Single element:
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.print_sorted()
    [1]

Multiple elements:
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]

Original list not modified:
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.print_sorted()
    [1, 2, 4]
    >>> print(my_list)
    [1, 4, 2]

Already sorted list:
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.print_sorted()
    [1, 2, 3]

Reverse sorted list:
    >>> my_list = MyList()
    >>> my_list.append(5)
    >>> my_list.append(4)
    >>> my_list.append(3)
    >>> my_list.append(2)
    >>> my_list.append(1)
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]

Negative numbers:
    >>> my_list = MyList()
    >>> my_list.append(-1)
    >>> my_list.append(-4)
    >>> my_list.append(-2)
    >>> my_list.print_sorted()
    [-4, -2, -1]

Mixed positive and negative numbers:
    >>> my_list = MyList()
    >>> my_list.append(-1)
    >>> my_list.append(4)
    >>> my_list.append(-2)
    >>> my_list.append(3)
    >>> my_list.print_sorted()
    [-2, -1, 3, 4]

Duplicate elements:
    >>> my_list = MyList()
    >>> my_list.append(3)
    >>> my_list.append(1)
    >>> my_list.append(3)
    >>> my_list.append(1)
    >>> my_list.print_sorted()
    [1, 1, 3, 3]

--- isinstance check ---

MyList is a subclass of list:
    >>> issubclass(MyList, list)
    True

Instance is also a list:
    >>> my_list = MyList()
    >>> isinstance(my_list, list)
    True
