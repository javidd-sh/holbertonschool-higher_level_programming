# Python - Integers Addition

This project contains a Python function that adds two integers.

## Function

`add_integer(a, b=98)`

* Adds two numbers
* Accepts integers and floats
* Floats are converted to integers
* Raises `TypeError` if arguments are not numbers

## Files

* `0-add_integer.py` – function implementation
* `tests/0-add_integer.txt` – test cases
