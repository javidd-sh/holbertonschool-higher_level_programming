# Python - Abstract Classes, Interfaces, and Duck Typing

## Description
This project covers advanced Object-Oriented Programming concepts in Python,
including abstract classes, interfaces, duck typing, and mixins. It explores
how to enforce structure in class hierarchies and write flexible, reusable code.

## Learning Objectives
- What is an abstract class and how to create one using the `ABC` module
- What is the `@abstractmethod` decorator and how to use it
- What is duck typing and how Python uses it
- What are interfaces and how to simulate them in Python
- What are mixins and how to use them to share functionality
- How to use `isinstance` and `issubclass` effectively
- How to enforce method implementation in subclasses
- How to use multiple inheritance in Python
